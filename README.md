# 🚗 Car Sales Analysis Dashboard (Excel + IBM Cognos)

## 📌 Project Overview

This project analyzes car sales and profit data to generate insights for a regional dealership manager.
It combines spreadsheet-based analysis with business intelligence tools to create clear and interactive visualizations.

The dataset is a modified subset from the IBM Accelerator Catalog, designed for analytical learning.

---

## 📊 Key Objectives

* Analyze **quantity sold by dealer**
* Track **profit trends over time**
* Compare **dealer performance by year**
* Perform **model-specific analysis (Hudson cars)**

---

## 🛠️ Tools & Technologies

* Microsoft Excel (Excel for Web)
* Pivot Tables & Charts
* IBM Cognos Analytics (Dashboarding & Reporting)

---

## 📈 Visualizations Created

### 1. Quantity Sold by Dealer ID

* Bar chart
* Sorted by sales volume
* Identifies top-performing dealers

---

### 2. Profit by Date and Model

* Line chart
* Shows trends over time
* Compares different models

---

### 3. Profit by Year and Dealer ID

* Column chart
* Highlights yearly dealer performance
* Styled with red columns

---

### 4. Profit of Hudson Models by Dealer ID

* Line chart
* Focused analysis on Hudson model
* Customized with:

  * Removed gridlines
  * Legend positioned on the right
  * Green line styling
